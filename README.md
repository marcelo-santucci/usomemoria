# 🎛️ Uso de Memoria

Revisar el uso de memoria por parte de un programa por parte del sistema operativo en tiempo de ejecución de manera más precisa.

+ usomemoria.c :- Despliega el valor del consumo de memoria del programa.

Al remover los comentarios y compilar nuevamente inicia el incremento del uso de memoria que puede ser monitoreado mediante el comando:

     > ps ux <br>
